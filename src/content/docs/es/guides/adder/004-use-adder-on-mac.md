# Instalación y uso en macOS

1. Descarga el archivo `.pkg` desde la página de versiones y elige el paquete que coincida con la arquitectura de tu Mac, como `arm64` o `amd64`.
2. Abre el instalador y completa los pasos para instalar Adder.
3. El instalador coloca `Adder.app` en `/Applications` y, cuando puede crear el enlace, deja `adder` disponible en `/usr/local/bin/adder`.
4. Abre `Adder.app` desde `/Applications` o ejecuta `adder` en Terminal.

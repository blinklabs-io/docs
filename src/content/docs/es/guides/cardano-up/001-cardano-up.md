Utilidad de linea de comandos para gestionar servicios de Cardano. cardano-up te permite usar una utilidad de linea de comandos para instalar servicios de Cardano utilizando imagenes de docker.


# Por que usar cardano-up?

Imagina que eres un desarrollador y para programar o probar tu aplicacion necesitas un nodo de Cardano y quizas Kupo u Ogmios. Con cardano-up puedes usar una sola linea de comandos para instalar cada uno de estos servicios como una imagen de docker. Deja de perder tiempo instalando y comienza a programar.

## Como uso cardano-up?
Se requiere Docker, asi que puedes instalar Docker o Docker Desktop, luego simplemente descarga el archivo cardano-up desde blinklabs.io.


Comienza con cardano-up usando Docker Desktop con nuestra guia de [Inicio Rapido](../002-quick-start-docker-desktop).

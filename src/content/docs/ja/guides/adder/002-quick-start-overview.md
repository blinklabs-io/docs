# クイックスタート

Adderは、Cardanoブロックチェーンをテーリングし、ユーザーが設定可能なフィルターに基づいて、各ブロックとトランザクションのイベントを発行するツールです。これらのイベントは他のイベントベースのツールチェーンで処理でき、Cardano向けの豊富でリアクティブなアプリケーションを構築できます。

基本的に、Adderを使用すると、関心のあるイベントを追跡し、希望する方法で通知を受け取ることができます。例えば、追跡したいウォレットアドレスに支払いが送信されたときに、デスクトップ通知を受け取ることができます。また、ステークプールオペレーター（SPO）がブロックを生成したときにDiscordで通知を受け取ることもできます。Adderは、関心のあるさまざまなCardanoブロックチェーンイベントを追跡し、それらが発生したときに通知します。

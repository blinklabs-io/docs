## v0.1.15 (June 8, 2026)

**Title:** Improve 32 bit target compatibility

**Date:** June 8, 2026

**Version:** v0.1.15

This release addresses a compatibility issue that affected 32 bit builds. Integrations can now run those builds correctly, while behavior on 64 bit systems remains unchanged.

### ✨ What's New

This patch release focuses on compatibility and correctness for affected builds rather than introducing a new feature.

### 🔧 Fixes

* Fixed **32 bit CEK compatibility:** Plutigo now handles `unConstrData` overflow checks correctly on 32 bit targets, so those builds work as expected while 64 bit systems keep the same behavior.

### 📋 What You Need to Know

This update matters for environments that rely on 32 bit builds, because those targets now run correctly without changing behavior on 64 bit systems. Existing 64 bit integrations do not require any changes.

### 🙏 Thank You

Thank you for using Plutigo.

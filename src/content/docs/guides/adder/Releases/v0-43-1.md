## v0.43.1 (July 23, 2026)

**Title:** More reliable API routing, macOS packaging, and release automation

**Date:** July 23, 2026

**Version:** v0.43.1

This release keeps Adder's public API endpoints aligned while moving the server onto Go's `net/http` stack. It also strengthens macOS packaging, release automation, and runtime dependencies.

### ✨ Highlights

* Improved **keep public API routes aligned during the server migration:** Adder now serves `/ping`, `/healthcheck`, `/events`, and the `/v1` FCM and QR routes on Go's `net/http` stack with structured logging, panic recovery, and safer streaming behavior.

* Refined **ship signed and notarized macOS installers:** Adder now builds `.pkg` installers with installer preinstall and postinstall behavior, and it publishes and verifies macOS assets more reliably.

### 💪 Improvements

* Updated **keep release and CI workflows current:** Adder now uses refreshed Go workflow actions plus updated upload and attestation tooling, which makes release automation more dependable.

* Strengthened **keep core dependencies current:** Adder now picks up updates to `go/sys`, `x/image`, `plutigo`, `gouroboros`, and `x/sys`, which helps harden decoding and runtime behavior while keeping builds current.

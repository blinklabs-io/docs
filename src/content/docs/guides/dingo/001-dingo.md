![dingo-logo](/dingo-logo-250.png)

A Cardano blockchain data node written in Go which actively participates in network communications on the Cardano blockchain using the Ouroboros Network Node-to-Node family of mini-protocols.  

⚠️ This is a work in progress and is currently under heavy development

<br>

## How do I actually use Dingo?

In this guide we will walk you through getting the Dingo binary, creating the `dingo.yaml` configuration file, bootstrapping the Dingo node using a Mithril snapshot, and using the Cardano CLI to interact with the Dingo node. Dingo ships with embedded Cardano network configuration files. Please note this guide will run the Dingo node using the Cardano Preview network.

A Cardano blockchain data node written in Go that uses the Ouroboros node to node mini protocol family.  

⚠️ This is a work in progress and is currently under heavy development

<br>


## How do I actually use Dingo?

This guide covers getting the Dingo binary, creating configuration, bootstrapping with Mithril, and using Cardano CLI with Dingo on the Cardano Preview network.  

✅ Start with the [Quick Start](../002-quick-start-overview) guide.

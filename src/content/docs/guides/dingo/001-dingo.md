![dingo-logo](/dingo-logo-250.png)

A Cardano blockchain data node written in Go which actively participates in network communications on the Cardano blockchain using the Ouroboros Network Node-to-Node family of mini-protocols.  

⚠️ This is a work in progress and is currently under heavy development

<br>

## How do I actually use Dingo?

In this guide we will walk you through getting the Dingo binary, downloading the Cardano configuration files and how to bootstrap the Dingo node using a Mithril snapshot. It also covers exposing the API services used by the local Blockfrost explorer example. We will also explore how to use the Cardano CLI to interact with the Dingo node. Please note this guide will run the Dingo node using the Cardano Preview network.  

After a Mithril restore, Dingo now keeps the imported stake snapshot state during startup when it is already there.

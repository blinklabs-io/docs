![dingo-logo](/dingo-logo-250.png)

A Cardano blockchain node written in Go which actively participates in network communications on the Cardano blockchain using the Ouroboros Network Node-to-Node family of mini-protocols.  

⚠️ Dingo is a work in progress and is currently under heavy development

<br>

## How do I actually use Dingo?

In this guide, we will walk you through getting the Dingo binary and how to bootstrap the Dingo node using a Mithril snapshot. We will also explore how to use the Cardano CLI to interact with the Dingo node. Please note that this guide will run the Dingo node using the Cardano Preview network.

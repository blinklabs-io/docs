## v0.61.3 (July 7, 2026)

**Title:** Updated build dependencies and imported Mithril mark handling

**Date:** July 7, 2026

**Version:** v0.61.3

This release keeps Dingo aligned with current build and runtime dependencies and fixes the imported Mithril mark eligibility check.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### 💪 Improvements

* Improved the build toolchain by moving the first Vite dependency to a newer version.

* Refined the build toolchain by moving the second Vite dependency to a newer version.

* Updated the Docker build workflow so image builds use the latest maintained action.

* Modernized the Docker publish workflow so release publishing uses the latest maintained action.

* Enhanced AWS SDK v2 modules so Dingo picks up current upstream maintenance fixes.

* Streamlined gRPC dependencies so Dingo stays aligned with upstream compatibility updates.

* Corrected Google API dependencies so Dingo keeps current API client behavior.

* Refreshed `@types/node` so Node.js type definitions stay current.

### 🔧 Fixes

* Fixed imported Mithril mark eligibility checks so Dingo skips the check for imported marks.

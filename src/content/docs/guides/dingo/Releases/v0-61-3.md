## v0.61.3 (July 7, 2026)

**Title:** Updated dependency versions and imported Mithril mark handling

**Date:** July 7, 2026

**Version:** v0.61.3

This release keeps Dingo aligned with current build and runtime dependencies and corrects imported Mithril mark handling.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### 💪 Improvements

* Improved the build toolchain by moving one Vite dependency to a newer version.

* Refined the build toolchain by moving a second Vite dependency to a newer version.

* Updated Docker build actions so image builds use the latest maintained workflow.

* Modernized Docker publish actions so release publishing uses the latest maintained workflow.

* Enhanced AWS SDK v2 modules so Dingo picks up upstream maintenance fixes.

* Streamlined gRPC dependencies so Dingo stays aligned with upstream compatibility updates.

* Corrected Google API dependencies so Dingo keeps current API client behavior.

* Refreshed `@types/node` so Node.js type definitions stay current.

### 🔧 Fixes

* Fixed imported Mithril mark eligibility checks so Dingo now skips the check for imported marks.

## v0.49.1 (May 28, 2026)

**Title:** Stronger preview network support, clearer swap examples, and steadier node behavior

**Date:** May 28, 2026

**Version:** v0.49.1

Hi folks! Here’s what we shipped in v0.49.1.

### ✨ What's New

* Added **explore Dingo powered swaps with a Preview network example:** The new DingoSwap example shows how to query pools, handle assets, build swap orders, and connect a browser wallet for SundaeSwap V3 activity on Preview.
* Introduced **serve merged Leios ranking and endorser data to client applications:** Dingo can now deliver merged ranking and endorser blocks over node to client sync, which expands access to Leios data for downstream tools.
* Expanded **persist reward state for richer ledger tracking:** Dingo now keeps reward state data available across runs, which improves continuity for reward and ada pot views.
* Enabled **apply reward account auto votes under CIP 1694 rules:** Stake pool reward account delegation now follows the updated governance semantics automatically.

### 💪 Improvements

* Enhanced **keep the Preview swap example aligned with current browser tooling:** The DingoSwap example now uses updated build tooling and wallet compatibility shims so local setup stays smoother.
* Updated **move Dingo builds onto Go 1.25:** The release now aligns test, publish, and module tooling on Go 1.25 for a more consistent build environment.

### 🔧 Fixes

* Corrected **return clearer transaction submission failures across eras:** Transaction submission errors now keep valid structured details, which makes rejection handling more reliable.
* Stabilized **surface chain sync send failures before connections drift out of view:** Dingo now reports asynchronous chain sync send failures through connection close events so operators can spot broken sessions sooner.
* Refined **prune header tracking during normal sync operation:** Dingo now trims retained header history during ongoing sync work, which helps long running nodes stay steadier over time.

| preview-testnet     | ✅         |

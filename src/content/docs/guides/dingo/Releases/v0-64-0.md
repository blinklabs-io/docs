## v0.64.0 (July 14, 2026)

**Title:** Improve pool imports and preserve governance balances through restarts

**Date:** July 14, 2026

**Version:** v0.64.0

This release simplifies pool imports and keeps governance accounting consistent when the node resumes at an epoch boundary.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Improved pool import workflows so library users can bring in pool state and registration data through one call, while Dingo starts the required metadata update when needed and stops invalid requests before any partial import begins.

* Corrected governance replay at the current epoch boundary so restarts preserve treasury withdrawals and expired proposal deposit refunds, while proposal searches by epoch and slot now return a consistent order across supported data stores.

## v0.50.1 (June 2, 2026)

**Title:** Strengthen blockfetch peer recovery, correct chain density, and reduce ledger recomputation

**Date:** June 2, 2026

**Version:** v0.50.1

Hi folks! Here’s what we shipped in v0.50.1.

### 💪 Improvements

* Improved **exclude sibling worktrees from routine Go file discovery:** Dingo now keeps local sibling worktrees from affecting formatting and build inputs, which makes routine maintenance work more dependable.

### 🔧 Fixes

* Strengthened **recover from stuck block downloads more safely:** Dingo now closes or slows peers that repeatedly fail to provide requested blocks, which helps sync work continue more reliably.
* Corrected **measure chain density more accurately:** Dingo now reports chain density in a way that better reflects the active chain fragment, which makes chain health signals more dependable.
* Reduced **avoid repeated next epoch nonce recomputation:** Dingo now skips unnecessary repeated nonce recalculation during slot updates, which reduces unneeded ledger work.

### 🧪 Examples, Docs, and Release Tooling

* Documented **publish the v0.50.0 release notes in the repository:** The repository release notes now include the previous version page, which keeps release history easier to review.

## v0.59.1

**Title:** Faster Mithril bootstrap downloads and steadier Leios backfill recovery

**Date:** June 27, 2026

**Version:** v0.59.1

Hi folks! Here’s what we shipped in v0.59.1.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Improved **speed up Mithril bootstrap downloads:** Dingo now reuses pooled keep alive HTTP connections across Mithril v2 immutable archive downloads and increases worker concurrency, which helps bootstrap jobs avoid repeated handshakes and complete more efficiently.

### 🔧 Fixes

* Stabilized **keep healthy Leios peers preferred during backfill:** Dingo now applies per connection cooldowns to failed leios fetch peers during backfill, which helps healthy peers stay preferred and reduces repeated retries against flaky connections.

## v0.59.1

**Title:** Faster Mithril bootstrap downloads and steadier Leios backfill recovery

**Date:** June 27, 2026

**Version:** v0.59.1

Hi folks! Here’s what we shipped in v0.59.1.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Improved **speed up Mithril bootstrap downloads:** Dingo now keeps Mithril bootstrap downloads moving through reused connections and more parallel archive work, which helps bootstrap jobs avoid repeated handshakes and finish more efficiently.

### 🔧 Fixes

* Stabilized **keep healthy Leios peers preferred during backfill:** Dingo now cools down failed Leios backfill connections individually, which helps healthy peers stay preferred and reduces repeated retries against flaky connections.

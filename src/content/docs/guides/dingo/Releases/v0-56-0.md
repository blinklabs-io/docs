## v0.56.0 (June 23, 2026)

**Title:** Improve Leios and Musashi compatibility, add checkpoint-based chain validation, and harden Mithril-assisted catch-up

**Date:** June 23, 2026

**Version:** v0.56.0

Hi folks! Here’s what we shipped in v0.56.0.

### ✨ Highlights

* Improved **keep Dingo aligned with newer Leios behavior:** Dingo now tracks newer Leios behavior more closely, which helps operators stay compatible with current preview network activity.

* Expanded **apply Leios endorsement blocks during chain processing:** Dingo now applies Leios endorsement blocks during chain processing, which improves compatibility with newer block flows on supported networks.

* Refined **validate chains against configured checkpoints:** Dingo now validates chain progress against checkpoints, which gives operators a clearer way to detect invalid history during sync.

* Enhanced **recognize Musashi naming across the product:** Dingo now uses Musashi naming consistently, which makes preview network configuration and operation clearer.

### 💪 Improvements

* Strengthened **show Mithril catch up progress more clearly:** Dingo now reports Mithril catch up progress more clearly, which makes sync recovery easier to follow.

* Improved **check genesis start times for consistent configuration:** Dingo now checks genesis start times for consistency, which helps operators catch mismatched network configuration earlier.

* Expanded **reduce repeated peer governance skip logging:** Dingo now reduces repeated peer governance skip logging, which makes operational logs easier to read during peer selection changes.

* Refined **prepare optional storage plugin builds more cleanly:** Dingo now includes build plumbing for optional storage plugin support, which makes tailored builds easier to prepare when that support is needed.

### 🔧 Fixes

* Fixed **rewind Mithril assisted catch up to a valid chain point:** Dingo now rewinds Mithril assisted catch up to a valid chain point, which helps ledger recovery continue cleanly after a rewind.

* Corrected **skip waits for outdated Leios endorsement blocks:** Dingo now skips waits for outdated Leios endorsement blocks, which helps chain progress continue when older blocks no longer matter.

## v0.63.1 (July 14, 2026)

**Title:** Reduce false error noise and refresh supporting dependencies

**Date:** July 14, 2026

**Version:** v0.63.1

This release reduces noise from expected duplicate connection shutdowns, refreshes bundled network configuration data, and keeps core dependencies and example applications current.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Refined expected duplicate connection shutdown logging so benign duplicate peer connection cleanup now records informational events instead of error noise.

* Updated bundled network configuration assets so tracer files and the Musashi network configuration stay current.

* Hardened concurrency safeguards in the updated sync stack so invalid negative weights now fail immediately instead of risking corrupted internal state.

* Refreshed service health dependencies while keeping existing Bark and UTxO RPC health checks wired the same way.

* Modernized low-level system libraries so Dingo stays aligned with current upstream OS support updates.

* Renewed cryptography, terminal, and text handling libraries so Dingo stays aligned with current upstream maintenance updates.

### 🧪 Testing, Docs, and Tooling

* Stabilized the Blockfrost explorer example by updating its frontend build toolchain.

* Advanced the Sundae preview example by updating its frontend build toolchain.

* Aligned the Sundae preview example with the current SundaeSwap SDK release.

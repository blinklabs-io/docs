## v0.46.3 (May 17, 2026)

**Title:** Stronger ChainSync recovery, faster database queries, and broader rollback test coverage

**Date:** May 17, 2026

**Version:** v0.46.3

Hi folks! Here’s what we shipped in v0.46.3.

### ✨ What's New

* Highlighted **no brand new end user features:** This patch release focuses on reliability, performance, testing coverage, and release note upkeep.

### 💪 Improvements

* Expanded **rollback coverage across more recovery scenarios:** Updated synthetic rollback checks now cover more conformance cases, which gives operators stronger confidence in rollback behavior.

### 🔧 Fixes

* Restored **safer post plateau peer recovery:** ChainSync now starts resync on a fresh connection after peer realignment, which makes post plateau recovery safer.

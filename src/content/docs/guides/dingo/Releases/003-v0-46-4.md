## v0.46.4 (May 18, 2026)

**Title:** Broader genesis bootstrap support, richer Mithril sync visibility, and sturdier database handling

**Date:** May 18, 2026

**Version:** v0.46.4

Hi folks! Here’s what we shipped in v0.46.4.

### ✨ What's New

* Expanded **bootstrap support beyond SQLite deployments:** Genesis bootstrap now works with MySQL and Postgres metadata stores, which gives more operators flexibility when they bring up new nodes from genesis data.
* Enhanced **Mithril sync visibility during recovery work:** Mithril sync now exposes progress and phase metrics through Prometheus, which makes it easier to track bootstrap, ledger import, block copy, gap filling, and backfill activity.

### 💪 Improvements

* Improved **faster SQLite work on UTxO heavy activity:** SQLite now uses a more efficient lookup path for transaction output references, which helps UTxO heavy processing complete more smoothly.
* Refined **keep S3 integrations current:** The AWS S3 library update keeps object storage support aligned with the latest maintained dependency set.
* Updated **keep Google API integrations current:** The Google API library update refreshes supporting integrations and maintenance coverage.
* Strengthened **keep Bitcoin utility support current:** The `btcutil` update refreshes supporting components that depend on Bitcoin utility handling.
* Clarified **keep recent release history easier to review:** The repository release notes now include the v0.46.3 entry, which makes recent patch history easier to follow.

### 🔧 Fixes

* Corrected **keep genesis bootstrap state consistent across metadata backends:** Bootstrap state now stays aligned more reliably across metadata plugins, which reduces the chance of inconsistent recovery behavior after bootstrap.
* Stabilized **prevent metadata label overflow in SQLite:** SQLite now handles very large transaction metadata labels correctly, which avoids failures when labels use high value ranges.

### 📋 What You Need to Know

* Highlighted **use broader database choices for bootstrap:** Operators can now bootstrap genesis data with MySQL and Postgres as well as SQLite.
* Emphasized **monitor Mithril sync progress more clearly:** Prometheus now exposes Mithril sync phases and progress so operators can see where sync time is going.
* Accelerated **expect faster SQLite handling for UTxO heavy workloads:** SQLite spends less time on transaction output lookups during heavy UTxO processing.
* Summarized **benefit from steadier maintenance fixes in this patch:** This release also improves bootstrap consistency, keeps key dependencies current, preserves recent release history, and fixes large metadata label handling in SQLite.

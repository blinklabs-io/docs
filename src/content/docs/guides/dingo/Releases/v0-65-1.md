## v0.65.1 (July 16, 2026)

**Title:** Restore Leios committee formation and keep relay recovery stable

**Date:** July 16, 2026

**Version:** v0.65.1

This release restores committee formation on supported Leios deployments when genesis files leave committee settings unset, keeps relay discovery and rollback recovery more reliable, and hardens validation around transaction submission, release verification, and reward metadata checks.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Restored committee formation on Musashi and other Dijkstra based Leios networks when genesis files leave committee settings unset, so certification can continue without changing a pinned genesis file while still rejecting invalid threshold combinations.

* Hardened local transaction submission handling so unexpected data fails safely instead of interrupting the submission path.

* Corrected single relay chain sync recovery so rollback processing can move forward instead of repeating the same loop.

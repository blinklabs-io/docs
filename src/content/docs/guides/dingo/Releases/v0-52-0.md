## v0.52.0 (June 8, 2026)

**Title:** Standalone history expiry and more accurate ledger fee validation

**Date:** June 8, 2026

**Version:** v0.52.0

Hi folks! Here’s what we shipped in v0.52.0.

### 💪 Improvements

* Improved **run history expiry without a separate archive fallback:** Dingo now lets operators expire older local block history on its own while keeping Bark archive fallback as a separate option, which makes storage planning more flexible for different deployments.

### 🔧 Fixes

* Corrected **validate ledger fees more accurately for Conway transactions:** Dingo now calculates minimum fees more accurately, including reference script costs and redeemer checks, which reduces the risk of transactions failing because fee validation was off.

## v0.65.0 (July 15, 2026)

**Title:** Preserve reward metadata and keep Leios catch up moving across slow peers

**Date:** July 15, 2026

**Version:** v0.65.0

This release keeps reward related database state more reliable, corrects pool stake totals used during validation, helps historical Leios catch up move past slow peers, and updates the example app development toolchains.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### ✨ Highlights

* Improved reward metadata handling so upgrades can rebuild missing live reward stake data at startup, keep reward state more consistent across supported databases, and retain more snapshot inputs for later reward processing without introducing reward calculation in this release.

* Corrected mark stake totals so pool stake snapshots include reward account balances at each slot, preserve delegators who only hold reward balances, and reduce stake threshold validation failures.

* Accelerated historical Leios catch up so Dingo moves to healthier peers when a relay responds slowly or stays busy, instead of stalling for long backfill waits.

* Updated the Sundae preview example development toolchain to use TypeScript 7.0.2 and require Node 16.20.0 or newer during local work.

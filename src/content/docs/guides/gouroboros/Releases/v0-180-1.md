## v0.180.1 (June 3, 2026)

**Title:** Align mint encoding for Conway and later validation

**Date:** June 3, 2026

**Version:** v0.180.1

Hi folks! Here’s what landed in v0.180.1.

### ✨ What's New

* Confirmed **no new features:** This patch release focuses on a validation fix and a release history documentation update.

### 💪 Improvements

* Corrected **mint encoding for Conway and later validation:** Plutus V1 and V2 script context mint encoding now matches cardano-ledger by including the zero ADA mint entry for PV10 and Conway and later protocol versions, which prevents CBOR mismatches during validation while preserving pre PV10 behavior.

### 📋 What You Need to Know

* Documented **release history maintenance during this window:** The upstream repository also published the `v0.180.0` release notes entry during this window, which updates release history documentation rather than runtime behavior.

Thank you for following gOuroboros releases.

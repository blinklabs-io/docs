## v0.188.1 (July 20, 2026)

**Title:** Improve leiosfetch handling, shutdown reliability, and withdrawal validation

**Date:** July 20, 2026

**Version:** v0.188.1

This release improves handling of missing leios fetch data, makes chain sync shutdown more reliable, tightens Conway withdrawal validation, and keeps the release archive current.

### 💪 Improvements

* Fixed `leiosfetch` so missing block or transaction requests now return not found responses instead of tearing down the connection.

* Improved `Stop` handling for chain sync and updated `plutigo` to `0.1.17` so shutdown now waits for completion instead of reporting `ErrProtocolShuttingDown` during teardown.

* Added Conway withdrawal validation for `PV10` and `PV11` so requests without `DRep` delegation now fail cleanly.

* Updated the Go workflow setup to `7.0.0` so build automation stays current.

* Published the `v0.187.4` release page so the earlier July 14 note now has a destination in the archive.

* Expanded the `v0.188.0` release page so the archive now includes the latest July 14 note.

### 🙏 Thank You

Thank you for following gOuroboros releases.

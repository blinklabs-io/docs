## v0.188.1 (July 20, 2026)

**Title:** Improve leiosfetch handling, shutdown reliability, withdrawal validation, and build tooling

**Date:** July 20, 2026

**Version:** v0.188.1

This release improves leiosfetch not found handling, makes chain sync shutdown more reliable, tightens Conway withdrawal validation, and updates build tooling.

### 💪 Improvements

* Fixed **leiosfetch not found responses:** missing block or transaction requests now return not found responses instead of tearing down the connection.

* Improved **chain sync shutdown:** `Stop` now waits for shutdown completion and suppresses `ErrProtocolShuttingDown` during teardown.

* Fixed **Conway withdrawal validation:** withdrawals on `PV10` and `PV11` now fail validation unless `DRep` delegation is present.

* Updated **build tooling:** the `plutigo` dependency bump and `actions/setup-go` pin update keep the release workflow current.

Thank you for following gOuroboros releases.

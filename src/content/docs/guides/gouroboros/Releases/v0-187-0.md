## v0.187.0 (July 6, 2026)

**Title:** Enforce reference script limits and align Leios block encoding

**Date:** July 6, 2026

**Version:** v0.187.0

This release tightens reference script validation and updates Leios block encoding and decoding to match the expected prototype-2026w27 format.

### 💪 Improvements

* Added **reference script size checks:** gOuroboros now rejects transactions and blocks that exceed the allowed reference-script size limits, using typed errors to distinguish per-transaction and per-block failures.

* Fixed **Leios block encoding for prototype-2026w27:** blocks carrying a Leios certificate now encode the expected empty Dijkstra transaction slots, and the decoder now rejects malformed or placeholder certificate payloads so Leios data stays consistent on the wire.

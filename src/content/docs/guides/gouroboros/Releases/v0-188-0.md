## v0.188.0 (July 14, 2026)

**Title:** Improve reply delivery confirmation and refresh project guidance

**Date:** July 14, 2026

**Version:** v0.188.0

This release improves reply delivery confirmation for Leios notifications and refreshes published project guidance to reflect current feature coverage, compatibility details, and testing status.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### 💪 Improvements

* Improved **Leios notification delivery confirmation:** services can now confirm whether a reply reached the connection, failed while sending, or stopped during shutdown, which makes retry and recovery decisions clearer.

* Refreshed **project status guidance:** the published project documentation now reflects current feature coverage, supported query areas, ledger compatibility details, and testing status without changing runtime behavior.

Thank you for following gOuroboros releases.

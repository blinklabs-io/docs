## v0.188.0 (July 14, 2026)

**Title:** Improve Leios notification delivery confirmation

**Date:** July 14, 2026

**Version:** v0.188.0

This release improves Leios notification delivery confirmation so producers can act on confirmed sends, and it refreshes repository status information in the published project documentation.

### Recommended Network Compatibility ⚠️

| Network             | Compatible |
|---------------------|------------|
| mainnet             | ⛔         |
| preprod-testnet     | ⛔         |
| preview-testnet     | ✅         |

### 💪 Improvements

* Improved **Leios notification delivery confirmation:** producers can now learn whether a response reached the connection, failed during transmission, or stopped during shutdown, which supports clearer retry and recovery decisions.

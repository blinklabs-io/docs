## v0.187.4 (July 14, 2026)

**Title:** Improve transaction submission shutdown and prototype vote compatibility

**Date:** July 14, 2026

**Version:** v0.187.4

This release improves transaction submission reliability, expands Leios vote compatibility, and updates the crypto packages used by builds and bundled examples.

### 💪 Improvements

* Fixed **clean transaction submission shutdown:** clients can now close the transaction submission session cleanly after a successful submission instead of triggering a protocol state error that disconnects the node client connection.

* Expanded **Leios prototype vote compatibility:** integrations now work with peers that send the current prototype vote format while continuing to accept older vote identifier and full vote formats. Invalid signature lengths now fail decoding immediately.

* Updated **crypto dependency resolution:** project builds and bundled examples now resolve `golang.org/x/crypto` v0.54.0 with matching `golang.org/x/sys` versions, bringing upstream fixes and hardening without changing project APIs.

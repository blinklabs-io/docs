## v0.187.4 (July 14, 2026)

**Title:** Improve transaction submission shutdown and prototype vote compatibility

**Date:** July 14, 2026

**Version:** v0.187.4

This release improves transaction submission reliability, expands Leios vote compatibility, and updates crypto dependencies used by builds and bundled examples.

### 💪 Improvements

* Fixed **LocalTxSubmission idle / done flow:** clients can now close LocalTxSubmission cleanly after a successful transaction submission instead of hitting a protocol state error that tears down the node to client connection.

* Expanded **Leios prototype vote wire format support:** integrations now interoperate with peers using the current prototype vote encoding while continuing to accept the existing two element vote ID and legacy four element full vote shapes. Malformed prototype votes with an invalid BLS signature length now fail decoding.

## v0.189.1 (July 23, 2026)

**Title:** Fix GetCBOR query decoding and stake snapshot pool filters

**Date:** July 23, 2026

**Version:** v0.189.1

This release fixes GetCBOR-wrapped Shelley query decoding and keeps stake snapshot pool filters in the wire format cardano-node expects.

### 💪 Improvements

* Fixed **GetCBOR-wrapped Shelley query decoding:** `cardano-cli query stake-snapshot` and similar local state query commands now decode Shelley queries instead of failing and closing the node-to-client connection.

* Corrected **stake snapshot pool filter handling:** stake snapshot requests now encode and decode the pool filter in the wire format cardano-node expects, so all-pools and specific-pool requests work as intended.

Thank you for following gOuroboros releases.

## v0.189.0 (July 22, 2026)

**Title:** Improve Byron EBB decoding and add opcert counter decoding

**Date:** July 22, 2026

**Version:** v0.189.0

This release fixes malformed Byron EBB payloads and adds typed LocalStateQuery decoding for operational certificate counters.

### 💪 Improvements

* Fixed **Byron EBB nil-header decoding:** `ByronEpochBoundaryBlock.UnmarshalCBOR` now rejects payloads that decode to a missing header, so malformed Byron EBB data fails early instead of creating a block that can panic later.

* Added **typed opcert counter decoding:** LocalStateQuery now decodes `DebugChainDepState` into operational-certificate counters and exposes them through `GetOpCertCounters`, making the counters available through the typed API.

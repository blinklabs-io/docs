## v0.187.1 (July 7, 2026)

**Title:** Strengthen genesis staking pool parsing and align Dijkstra block handling

**Date:** July 7, 2026

**Version:** v0.187.1

This release improves genesis staking pool parsing and aligns Dijkstra and prototype-2026w27 block handling with the ledger specification.

### 💪 Improvements

* Fixed **genesis staking pool aliases:** the parser now accepts Shelley genesis names `publicKey`, `vrf`, `owners`, and `metadata` alongside the existing on-chain and API field names, which keeps the VRF key hash attached to genesis-declared pools.

## v0.186.3 (July 6, 2026)

**Title:** Tolerate Conway witness duplicates and refresh the lint workflow

**Date:** July 6, 2026

**Version:** v0.186.3

This release tolerates duplicate Conway witness members in historical blocks and refreshes the GitHub Actions lint workflow.

### 💪 Improvements

* Fixed **Conway witness decoding for tolerated tag-258 fields:** `ConwayTransactionWitnessSet.UnmarshalCBOR` now tolerates duplicate members in vkey witnesses, bootstrap witnesses, native scripts, and plutus data for Conway protocol versions 9 through 11, so historical Conway blocks no longer fail with `duplicate member in set` for those fields while Plutus V1, V2, and V3 script sets still reject duplicates.

* Updated **the GitHub Actions lint workflow:** `golangci/golangci-lint-action` now runs at version `9.3.0` instead of `9.2.1`, which changes lint output and behavior in CI only.

### 🙏 Thank You

Thank you for following gOuroboros releases.

## v0.189.2 (July 24, 2026)

**Title:** Document the latest release notes and fix Conway zero withdrawals

**Date:** July 24, 2026

**Version:** v0.189.2

This release adds the v0.189.0 and v0.189.1 release note updates and fixes Conway withdrawal validation for zero-amount entries.

### 💪 Improvements

* Added **v0.189.0 release notes:** The documentation now includes the v0.189.0 release note entry covering opcert counter decoding, Byron EBB validation, and related maintenance updates.

* Expanded **v0.189.1 release notes:** The documentation now includes the v0.189.1 release note entry covering local state query decoding and stake snapshot pool filters.

* Fixed **Conway zero withdrawal validation:** Transactions that contain only zero-amount withdrawals no longer fail DRep delegation checks, and mixed withdrawals validate only the non-zero entries.

### 🙏 Thank You

Thank you for following gOuroboros releases.

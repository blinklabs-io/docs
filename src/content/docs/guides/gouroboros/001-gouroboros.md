gOuroboros is a Go framework for building applications that interact with Cardano nodes, blocks, and transactions. It supports syncing the blockchain from a local or remote node, querying local node data such as protocol parameters and UTxOs by address, and using LocalStateQuery to access ledger peer snapshot data.

***

Learn more about the code documentation of gOuroboros here: <a href="https://pkg.go.dev/github.com/blinklabs-io/gouroboros" target="_blank">https://pkg.go.dev/github.com/blinklabs-io/gouroboros</a>.

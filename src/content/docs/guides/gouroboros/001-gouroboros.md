gOuroboros is a powerful and versatile framework for building Go apps that interact with the Cardano blockchain. It helps developers communicate with Cardano nodes, manage blocks and transactions, and inspect CBOR, Cardano transactions, and Cardano blocks with Cardano-aware diagnostic output and programmatic diagnostic APIs. The repository also includes first-party example applications for common workflows such as chain sync, block fetch, peer sharing, state query, transaction monitoring, and transaction submission under `examples/`.

***

Learn more about the code documentation of gOuroboros here: <a href="https://pkg.go.dev/github.com/blinklabs-io/gouroboros" target="_blank">https://pkg.go.dev/github.com/blinklabs-io/gouroboros</a>.
